EV Safety Systems

Tractive System Active Light (TSAL)

Project Overview

Designed, optimized, and built a custom Tractive System Active Light system tailored for Electric Vehicles. This hardware architecture delivers deep immersion into advanced discrete analog circuit topologies and precise PCB development constraints to explicitly track and indicate live battery accumulator high-voltage states.

State Indication Logic Breakdown

  • 🔴 Active Hazard Notification: Outputs a highly visible red blinking light state whenever live bus potentials cross beyond the critical 60V envelope.
  • 🟢 Safe Status Configuration: Actuates a solid green indicator lamp output whenever voltage drops safely beneath the 60V threshold limit.
  • ⚫ Single Wire Fault Safe-State: Automatically drops all visual indicators to a complete dark zero-power state during system failures to isolate hazardous false-positives.

Architecture Redesign & Optimization Metrics

Transitioned the classic safety architecture into an optimized Op-Amp comparator plus 555-Timer oscillation topology embedded with single-wire fault protection loops. This architectural design choice achieved key efficiency goals:

~15%Faster Response Speed
~25%PCB Size Reduction

Formula Bharat Compliance

Brake System Plausibility Device (BSPD)

Project Overview

Designed, assembled, and validated a robust two-layer Brake System Plausibility Device (BSPD) circuit board using Altium Designer for Team Kratos Racing Electric India. The safety asset continuously monitors concurrent pedal telemetry errors to preserve cockpit mechanical integrity.

Functional & Regulatory Specifications

  • Automatic Safety Cutoff: Opens the master vehicle shutdown path automatically if motor power yields over 5 kW simultaneously while hydraulic brake lines pass 30 bar.
  • Formula Bharat Rulebook Compliance: Features direct internal current distribution from the Low Voltage Master Switch (LVMS) with minimal auxiliary hardware line splits.
  • Sensor Signal Integration: Deploys specialized configurations to capture differential inputs from physical analog brake transducers and primary DC current tracks.

Engineering Challenges & Solutions

1. Component Placement & Densities: Spatial boundary guidelines imposed complex routing bottlenecks. Optimized trace paths using iterative layout evaluations to preserve safe high-voltage spatial boundaries.
2. Trace Routing & Signal Integrity: Analog trace noise threatened rulebook compliance. Leveraged high-precision track shielding loops and verified signal cleanliness via extensive LTspice simulation profiles.

Topological Optimization Improvements

Replaced obsolete digital logic arrays with an optimized Op-Amp comparator plus discrete MOSFET safety circuit path to maximize response stability.

35%PCB Footprint Reduction
18%Response Delay Speedup

Project Overview

Programmed and optimized the primary core embedded system firmware for an STM32-based automotive Electronic Control Unit (ECU). The implementation manages multiple high-speed vehicular peripheral buses, processing core control loop algorithms dynamically.

Technical Architecture Breakdown

  • Fault Diagnostic Loops: Developed hardware-software error detection algorithms to support debugging, troubleshooting, and rapid isolation of vehicle electronics faults.
